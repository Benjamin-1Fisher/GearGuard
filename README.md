# GearGuard Event Command Center

A hackathon MVP for a real-time equipment accountability system at large campus events.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite.

## Demo flow

1. Open the dashboard and notice the event is not ready to close.
2. Check out an available critical item from the warehouse.
3. Mark an item missing in the Return Ritual.
4. Open Rescue Mode and create a rescue task.
5. Return the item as OK.
6. Return the remaining critical items.
7. Watch the event become ready to close.
